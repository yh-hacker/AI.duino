## 安装
### Windows（推荐）
1. 右键点击“install_windows.bat”并选择“以管理员身份运行”
2. 按照屏幕上的指示进行操作
3. 重启Arduino集成开发环境（IDE）
### Linux
```bash
chmod +x install_aiduino_linux.sh
./install_aiduino_linux.sh
```
### macOS
```bash
chmod +x install_aiduino_macos.sh
./install_aiduino_macos.sh
```
## 卸载
### Windows（推荐）
1. 右键点击“uninstall_aiduino_windows”并选择“以管理员身份运行”
2. 按照屏幕上的指示进行操作
3. 重启Arduino集成开发环境（IDE）
### Linux
```bash
chmod +x uninstall_aiduino_linux.sh
./uninstall_aiduino_linux.sh
```
### macOS
```bash
chmod +x uninstall_aiduino_macos.sh
./uninstall_aiduino_macos.sh
